# Cyber Attacks on AI ⚠️

## Adversarial ML 🧠
- Techniques used to fool machine learning models by adding subtle noise to inputs.
- Causing the models to make incorrect predictions.

## Prompt Injection 💬
- Bypassing LLM filters by providing specifically crafted prompts.
- Force the AI to execute unintended commands or leak sensitive data.

## Data Poisoning 🧪
- Corrupting the training data of an AI model to create backdoors.
- Degrade the performance of the system for specific targets.
