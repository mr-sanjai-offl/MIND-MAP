# Sanjai's_Network: Cyber Security Insights

A modern, high-performance mind map CMS built with Vite, TypeScript, and Express. Projections for 2026.

## 🚀 One-Command Run
This project uses a unified server to serve both the frontend and the Markdown API.

1. **Install dependencies**:
   ```bash
   npm install
   ```
2. **Run Production**:
   ```bash
   npm run serve
   ```
   *This will build the frontend and start the server at `http://localhost:3001`.*

3. **Run Development**:
   - Frontend: `npm run dev`
   - Backend: `npm run backend`

## 📝 Markdown CMS Guide
All mind maps are managed as `.md` files in the `/content` directory.

### Structure:
- `# Root Topic`: The central node of the mind map.
- `## Category Name`: Major branches originating from the root.
- `- Topic Detail`: Individual nodes within a category.

### Example:
```markdown
# AI in Cyber 🧠
## Threat Detection 🔍
- Anomaly Models. Uses LSTM to find patterns.
- Real-time Scanning. Latency under 10ms.
```

## 🛠️ Features
- **Dynamic Selection**: Pick topics from a premium home screen.
- **Custom Mind Map Engine**: Interactive SVG curves and nodes.
- **Particle System**: Cyber-themed background effects.
- **No Database**: 100% file-system based.
